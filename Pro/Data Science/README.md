# Data Science - Pro Level

## Overview

Welcome to the Pro Data Science track! This level focuses on advanced analytics, causal inference, and large-scale data science solutions.

## Learning Objectives

- Implement advanced statistical and causal inference methods
- Build production-scale data science systems
- Apply advanced feature engineering and selection
- Master end-to-end data science workflows

## Requirements

- Python 3.x
- numpy
- matplotlib
- Any other libraries you choose to use

## Tasks

Complete **ANY TWO** of the three tasks described in `tasks.md`.

## Submission

Complete your chosen tasks in the `starter.ipynb` notebook and submit according to the main repository guidelines.
