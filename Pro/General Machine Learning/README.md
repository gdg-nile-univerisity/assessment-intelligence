# General Machine Learning - Pro Level

## Overview

Welcome to the Pro Machine Learning track! This level covers advanced ML systems, AutoML, and production-grade implementations.

## Learning Objectives

- Build complex ML systems and pipelines
- Implement AutoML and neural architecture search
- Master advanced ensemble and meta-learning techniques
- Deploy scalable ML solutions

## Requirements

- Python 3.x
- numpy
- matplotlib
- Any other libraries you choose to use

## Tasks

Complete **ANY TWO** of the three tasks described in `tasks.md`.

## Submission

Complete your chosen tasks in the `starter.ipynb` notebook and submit according to the main repository guidelines.
