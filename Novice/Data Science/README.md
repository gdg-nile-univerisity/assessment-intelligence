# Data Science - Novice Level

## Overview

Welcome to the Data Science track! This level introduces you to fundamental data analysis and predictive modeling concepts.

## Learning Objectives

- Understand basic data analysis workflows
- Learn to work with structured data
- Apply simple predictive modeling techniques
- Gain hands-on experience with data exploration

## Requirements

- Python 3.x
- numpy
- matplotlib
- Any other libraries you choose to use

## Tasks

Complete **ANY TWO** of the three tasks described in `tasks.md`.

## Submission

Complete your chosen tasks in the `starter.ipynb` notebook and submit according to the main repository guidelines.
