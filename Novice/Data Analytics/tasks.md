# Data Analytics Tasks - Novice Level

Complete **ANY TWO** of the following tasks.

---

## Task 1: Sales Data Analysis

### Objective
Analyze sales data to extract business insights.

### Requirements
1. Create or load a sales dataset with:
   - Date, product, quantity, price, region
2. Perform the following analyses:
   - Calculate total revenue by product/region
   - Identify best and worst performing products
   - Analyze sales trends over time
   - Calculate growth rates
3. Create visualizations for key metrics
4. Provide actionable business recommendations
5. Summarize findings in a clear format

### Hints
- Use pandas for data manipulation
- Group data by different dimensions
- Calculate percentage changes
- Create time series visualizations

---

## Task 2: Customer Behavior Analysis

### Objective
Analyze customer behavior patterns and segments.

### Requirements
1. Create or load customer data with:
   - Customer ID, purchase history, demographics
2. Perform the following analyses:
   - Calculate customer lifetime value
   - Identify customer segments
   - Analyze purchase frequency
   - Find patterns in customer behavior
3. Create visualizations showing segments
4. Develop customer personas
5. Suggest retention strategies

### Hints
- Use aggregation functions
- Consider RFM analysis (Recency, Frequency, Monetary)
- Visualize distributions
- Look for natural groupings

---

## Task 3: Simple Statistical Analysis

### Objective
Apply statistical methods to answer business questions.

### Requirements
1. Choose a business problem to analyze
2. Perform the following statistical analyses:
   - Descriptive statistics
   - Hypothesis testing (t-test or chi-square)
   - Correlation analysis
   - Confidence intervals
3. Interpret results in business context
4. Create visualizations supporting your analysis
5. Document assumptions and limitations

### Hints
- Use numpy/scipy for statistical functions
- Check assumptions before tests
- Visualize distributions and relationships
- Explain p-values and significance

---

## Evaluation Criteria

- **Correctness**: Analysis produces valid results
- **Code Quality**: Clean, readable, well-commented code
- **Documentation**: Clear explanations and business insights
- **Practicality**: Actionable recommendations
