# Data Analytics - Novice Level

## Overview

Welcome to the Data Analytics track! This level introduces you to fundamental data analysis techniques and business analytics.

## Learning Objectives

- Understand basic data analytics workflows
- Learn to extract insights from data
- Apply statistical analysis techniques
- Gain hands-on experience with business problems

## Requirements

- Python 3.x
- numpy
- matplotlib
- Any other libraries you choose to use

## Tasks

Complete **ANY TWO** of the three tasks described in `tasks.md`.

## Submission

Complete your chosen tasks in the `starter.ipynb` notebook and submit according to the main repository guidelines.
