# Natural Language Processing - Novice Level

## Overview

Welcome to the Natural Language Processing track! This level introduces you to fundamental text processing and analysis techniques.

## Learning Objectives

- Understand basic text preprocessing techniques
- Learn to work with text data using Python
- Apply simple NLP techniques for text analysis
- Gain hands-on experience with text classification

## Requirements

- Python 3.x
- numpy
- matplotlib
- Any other libraries you choose to use

## Tasks

Complete **ANY TWO** of the three tasks described in `tasks.md`.

## Submission

Complete your chosen tasks in the `starter.ipynb` notebook and submit according to the main repository guidelines.
