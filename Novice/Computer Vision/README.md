# Computer Vision - Novice Level

## Overview

Welcome to the Computer Vision track! This level introduces you to fundamental concepts in image processing and computer vision using Python.

## Learning Objectives

- Understand basic image operations and transformations
- Learn to work with image data using numpy and other libraries
- Apply simple filters and techniques for image processing
- Gain hands-on experience with real-world computer vision tasks

## Requirements

- Python 3.x
- numpy
- matplotlib
- Any other libraries you choose to use

## Tasks

Complete **ANY TWO** of the three tasks described in `tasks.md`.

## Submission

Complete your chosen tasks in the `starter.ipynb` notebook and submit according to the main repository guidelines.
