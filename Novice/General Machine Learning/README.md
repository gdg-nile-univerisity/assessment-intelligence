# General Machine Learning - Novice Level

## Overview

Welcome to the General Machine Learning track! This level introduces you to fundamental ML concepts and algorithms.

## Learning Objectives

- Understand basic machine learning concepts
- Learn to implement simple ML algorithms from scratch
- Apply data preprocessing and evaluation techniques
- Gain hands-on experience with classification and regression

## Requirements

- Python 3.x
- numpy
- matplotlib
- Any other libraries you choose to use

## Tasks

Complete **ANY TWO** of the three tasks described in `tasks.md`.

## Submission

Complete your chosen tasks in the `starter.ipynb` notebook and submit according to the main repository guidelines.
