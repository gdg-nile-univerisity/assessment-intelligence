# Deep Learning - Novice Level

## Overview

Welcome to the Deep Learning track! This level introduces you to fundamental neural network concepts and practical implementations.

## Learning Objectives

- Understand basic neural network architecture
- Learn to implement simple networks using numpy
- Apply activation functions and backpropagation
- Gain hands-on experience with training neural networks

## Requirements

- Python 3.x
- numpy
- matplotlib
- Any other libraries you choose to use

## Tasks

Complete **ANY TWO** of the three tasks described in `tasks.md`.

## Submission

Complete your chosen tasks in the `starter.ipynb` notebook and submit according to the main repository guidelines.
