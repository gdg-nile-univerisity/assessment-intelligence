# Deep Learning - Intermediate Level

## Overview

Welcome to the Intermediate Deep Learning track! This level covers neural network architectures, optimization techniques, and practical training strategies.

## Learning Objectives

- Implement multi-layer neural networks
- Apply regularization and optimization techniques
- Work with convolutional and recurrent architectures
- Handle real-world deep learning challenges

## Requirements

- Python 3.x
- numpy
- matplotlib
- Any other libraries you choose to use

## Tasks

Complete **ANY TWO** of the three tasks described in `tasks.md`.

## Submission

Complete your chosen tasks in the `starter.ipynb` notebook and submit according to the main repository guidelines.
