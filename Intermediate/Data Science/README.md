# Data Science - Intermediate Level

## Overview

Welcome to the Intermediate Data Science track! This level covers advanced analytics, predictive modeling, and machine learning applications for business problems.

## Learning Objectives

- Build end-to-end ML pipelines
- Apply advanced feature engineering
- Work with real-world datasets
- Communicate insights effectively

## Requirements

- Python 3.x
- numpy
- matplotlib
- Any other libraries you choose to use

## Tasks

Complete **ANY TWO** of the three tasks described in `tasks.md`.

## Submission

Complete your chosen tasks in the `starter.ipynb` notebook and submit according to the main repository guidelines.
