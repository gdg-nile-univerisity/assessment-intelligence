# Natural Language Processing - Intermediate Level

## Overview

Welcome to the Intermediate NLP track! This level covers advanced text processing, classification, and sequence modeling techniques.

## Learning Objectives

- Implement advanced text representation methods
- Build text classification models
- Work with word embeddings
- Apply sequence modeling techniques

## Requirements

- Python 3.x
- numpy
- matplotlib
- Any other libraries you choose to use

## Tasks

Complete **ANY TWO** of the three tasks described in `tasks.md`.

## Submission

Complete your chosen tasks in the `starter.ipynb` notebook and submit according to the main repository guidelines.
