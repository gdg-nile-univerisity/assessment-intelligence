# Data Analytics - Intermediate Level

## Overview

Welcome to the Intermediate Data Analytics track! This level covers advanced analytics techniques, cohort analysis, and optimization problems.

## Learning Objectives

- Perform advanced statistical analysis
- Conduct cohort and funnel analysis

- numpy
- matplotlib
