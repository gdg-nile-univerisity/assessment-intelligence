# Computer Vision - Intermediate Level

## Overview

Welcome to the Intermediate Computer Vision track! This level builds on fundamental concepts with more advanced techniques in image processing and object detection.

## Learning Objectives

- Implement advanced image processing algorithms
- Learn feature extraction and matching techniques
- Apply image segmentation methods
- Work with real-world computer vision challenges

## Requirements

- Python 3.x
- numpy
- matplotlib
- Any other libraries you choose to use

## Tasks

Complete **ANY TWO** of the three tasks described in `tasks.md`.

## Submission

Complete your chosen tasks in the `starter.ipynb` notebook and submit according to the main repository guidelines.
